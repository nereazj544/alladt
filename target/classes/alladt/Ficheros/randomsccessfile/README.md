# RandomAccessFile 
Es importante saber que cada tipo de variable ocupa un numero de bytes.

- Long: 8 bytes
- Integer (int): 4 bytes
- Short: 2 bytes
- Byte: 1 byte
- Double: 8 bytes
- Float: 4 bytes
- Boolean: 1 byte
- Char: 2 bytes
- String: 2 bytes por cada caracter.
